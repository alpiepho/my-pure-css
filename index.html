My Pure CSS Challenge

This is the set of daily challenges from the Daily CSS Challenge, 50 days worth of projects to build images and animations from HTML and CSS only...no Javascript. It's amazing what can be done with CSS alone.

Each challenge focuses on clean code and a different HTML/CSS technology. I completed these in the summber of 2017, but I remain a **HUGE** fan of CSS.
